Your life, organized.
On your machine.

LifeFlow is a fast, private, local-first task manager. Goals, focus timer, habits, weekly reviews — zero cloud, zero subscription.

Open LifeFlow Getting Started

Everything you need. Nothing you don't.

🎯

Life Areas & Goals

Organize tasks under life areas (Work, Health, Personal) and track goals with milestones.

⏱️

Focus Timer

Pomodoro-style focus sessions tied to your tasks. Plan, focus, reflect — all in one flow.

📊

Dashboard & Charts

Completion trends, streaks, area breakdowns, and weekly progress at a glance.

🔄

Recurring Tasks

Daily, weekly, biweekly, custom patterns. JSON config for power users. Auto-spawn on completion.

📅

Multiple Views

Today, All Tasks, Board, Calendar, Matrix, Weekly Plan — switch views with a keypress.

⌨️

Keyboard First

Vim-style navigation, rebindable shortcuts, command palette. Stay in the zone.

🏆

Badges & Streaks

Earn badges for milestones. Track your streak. Share focus cards with friends.

🔒

100% Local

SQLite database on your machine. No accounts, no cloud, no tracking. Your data is yours.

Get Started in 30 Seconds

1

Clone & Install

git clone https://github.com/user/lifeflow && cd lifeflow && npm install
2

Start the Server

npm start
3

Open in Browser

http://localhost:3000

Requires Node.js 18+. No external services needed.

Keyboard Shortcuts

Ctrl+K Search
N Quick add
1-9 Switch views
J/K Navigate
X Complete task
R Daily review
M Multi-select
? All shortcuts
